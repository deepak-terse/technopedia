## Prerequisites
*  Flutter
*  Dart
*  Android Basics (optional)

## Build Setup

``` bash
# clone the project
git clone https://github.com/deepak-terse/it-wasnt-me.git
cd it-wasnt-me

# install dependencies
flutter pub get

# run the application
flutter run

# hot reload
Press r after flutter run
```
